# zstack_stack_template (Resource)

Manages a stack template in ZStack.

`template_content` is required and must contain the `ZStackTemplateFormatVersion` marker. The provider validates this before calling the ZStack API.

## Example Usage

```terraform
# Copyright (c) ZStack.io, Inc.

resource "zstack_stack_template" "example" {
  name = "example-stack-template"
  template_content = jsonencode({
    ZStackTemplateFormatVersion = "2018-06-18"
    Resources                   = {}
  })
}

output "zstack_stack_template" {
  value = zstack_stack_template.example
}
```
